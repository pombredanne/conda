Build a package from a (conda) recipe.

(ADVANCED) For examples of recipes, see: https://github.com/ContinuumIO/conda-recipes

**usage**: ``conda build [-h] [--no-binstar-upload] [-s] [-t] PATH [PATH ...]``

*PATH*
        path to recipe directory

optional arguments:
  -h, --help           show this help message and exit
  --no-binstar-upload  do not ask to upload the package to binstar
  -s, --source         only obtain the source (but don't build)
  -t, --test           test package (assumes package is already build)
